## ğŸ†˜ TROUBLESHOOTING

### Site not showing up?
1. Check the green checkmark in Settings > Pages
2. Wait 5 minutes and refresh
3. Check your repository name is exactly "Carnival"

### Site shows 404 error?
1. Make sure index.html is in the root folder
2. Check file names match exactly (lowercase)
3. Go to Settings > Pages and verify settings

### Styles or JavaScript not loading?
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+Shift+R)
3. Check file names in HTML <link> and <script> tags

### Links not working?
1. Check you're using relative paths
2. Ensure all files are in the root directory
3. No capital letters in file names
